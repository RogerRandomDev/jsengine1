  • OBJECTS:
    • Object{
      • variables
        • x,y
          Dictates position
        • children[]
          the children nodes added with add(node)
        • parent:object
          the parent of the current object
        • visible:T/F
          decides whether it and its children will show and update
        • active:T/F
        • onclick:String
          will run the given string when clicked,
          effective when using string to trigger a function.
        • onhover:String
          works identically to onclick,
          but triggers whenever hovered instead
        • Size:vector2
          determines the Size of the object.
          rarely used
      • Functions
      • add(node)
        • adds the node as a child to self
      • moveWithCollision(dist(x,y))
        • takes a vector2 and must be in int
        • will cap motion to the screen itself and only checks with tilemap
      • checkhover()
          checks if you are hovered
          done automatically

        Variables

        • press()

        }
      tilemap extends object{
        Variables
        • tiles:Array
            Array of the tile images
        • map:Array
            16x16 Array of tile id's, -1 is empty
        Functions
        • setTile(index,name)
            sets tile at index to loadtex of name if index exists
        • addTile(name)
            puts tile at end of tiles array of the image loadtex(name)
        • setCell(x,y,id)
            sets map tile at position clamped to map size to selected id
        • updateMap()
            rebuilds map image, best used if you update map outside setCell
        • checkcollision(x,y)
            gets if there is something at the cell closest to x and y
        }
    }
